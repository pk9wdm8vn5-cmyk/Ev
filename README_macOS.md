# Пакет для macOS

| Файл | Открыть в |
|------|-----------|
| `Plan_pogasheniya_kreditov_macOS.pdf` | **Просмотр** (Preview) — Вид → Оглавление |
| `Plan_pogasheniya_kreditov_macOS.xlsx` | **Numbers** или Excel |
| `Plan_pogasheniya_kreditov_macOS.rtf` | **TextEdit** |
| `Plan_pogasheniya_kreditov_macOS.html` | **Safari** |
| `Plan_platezhey_iOS.ics` | **Календарь** (двойной клик → Добавить) |

## Быстрый старт на Mac
1. Дважды кликните PDF — читайте в Просмотре.
2. Откройте XLSX в Numbers — меняйте суммы на листе «Бюджет месяца».
3. Импортируйте ICS в Календарь.
4. HTML можно закрепить в Safari или добавить в Dock через Файл → Добавить в Dock (если доступно) / Избранное.
