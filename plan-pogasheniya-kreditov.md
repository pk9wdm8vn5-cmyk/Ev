## 6. Главный ускоритель: рефинансирование карт

Чтобы **остановить проценты** по картам, нужно разово:

| Карта | Сумма для остановки % |
|-------|----------------------:|
| Т-Банк (грейс) | 484 427 |
| Сбер | 417 717 |
| **Вместе** | **≈ 902 000** |

Это реально только через **рефинансирование / новый кредит**.

### Если рефинансировать обе карты в кредит @ ~20%

Сумма ≈ **919 000 ₽**

| Срок | Платёж/мес | vs сейчас по картам (мин 55к) |
|------|----------:|-------------------------------|
| 36 мес | ~34 200 | экономия по платежу |
| **48 мес** | **~28 000** | **сильно легче бюджету** |
| 60 мес | ~24 400 | ещё легче, но дороже переплата |

Эффект:

- минимумы по всем долгам падают примерно со **113к → ~86к**
- на процентах по «бывшим картам» экономия **~7 000 ₽/мес**
- появляется запас на жизнь **60–70к** без срыва минимумов
- полное закрытие всех долгов: **~24–32 мес** в зависимости от жизни

**Имеет смысл**, если одобрят ставку **≤ 22–24%**.  
Если предложат ≥ 25–28% — выгоды мало, лучше гасить avalanche’ем как есть.

### Полный рефинанс всего (~2 млн)

Платёж ~50–60к на 4–5 лет при 18–20% — **удобнее психологически**, но:

- переплата за 5 лет может превысить **1 млн ₽**;
- имеет смысл только при ставке **≤ 18–19%** и желании одного платежа.

## 7. Чего НЕ делать

1. **Не тратить** с кредитных карт (даже «чуть-чуть») — грейс уже сорван / проценты идут.  
2. **Не досрочно гасить** Сбер Б / Сбер В / Сбер А, пока живы карты и Т-кредит 19,9%.  
3. **Не платить** по картам только минимумы годами — переплата огромная.  
4. **Не брать** новый потребкредит «на жизнь» под высокую ставку.  
5. **Не игнорировать** обязательные даты — просрочка + штрафы съедят весь эффект плана.

## 8. Оптимальная стратегия (итог)

### Вариант A — без рефинанса (базовый)

1. Жёсткий бюджет жизни **45–50к**.  
2. Все минимумы вовремя.  
3. Любой рубль сверху → **Т-Банк карта → Сбер карта → Т-кредит → Сбер А**.  
4. После закрытия Сбер Б (~6 мес) его 15к автоматически добавить к досрочке по картам.  
5. Цель: **выход из долгов за ~2 года**.

### Вариант B — с рефинансом карт (предпочтительный, если дадут ≤22%)

1. Рефинансировать **обе карты** (~900к) на 36–48 мес @ ≤20–22%.  
2. Остальные кредиты — по графику.  
3. Досрочку направлять на рефинанс и/или Т-Банк кредит 19,9% (что дороже).  
4. Бюджет жизни можно держать комфортнее (**60к**).  
5. Цель: **~2–2,5 года**, меньше стресса и риска просрочки.

### Вариант C — гибрид (если рефинанс только на одну карту)

Приоритет рефина: **та карта, где выше ставка / больше %/мес**.  
Скорее Т-Банк (28%, долг больше) или обе, если лимит позволяет.
